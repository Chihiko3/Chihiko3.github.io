Yuzu Installation

Yuzu模拟器安装说明


模拟器本体

前往官网下载模拟器本体

  • "Download for Windows x 64"是下载一个安装器之类的软件（大概是为了让付费版自动更新更方便设计的）
  • "Builds"是直接下载一个文件压缩包（推荐，对于免费版来说够了）
  • 此时打开模拟器会报错，先忽视

Firmwares

前往官网下载最新Firmwares

  • MEGA和Archive.org是两种不同的下载方式，都可以
  • 不需要下载“Official China Firmwares”中的东西

Prod.keys

前往官网下载与Firmwares版本对应的Prod.keys

  • 如果网站过期了，直接Google搜索"prod keys"即可

解决模拟器报错

  • 放置Prod.keys
    • 模拟器左上角，“文件” -> “打开yuzu文件夹”
    • 将Prod.keys文件放入keys文件夹
  • 放置Firmwares
    • 模拟器左上角，“文件” -> “打开yuzu文件夹”
    • 将firmwares的文件全部解压到registered文件夹
    • ...\yuzu\nand\system\Contents\registered

安装游戏

  • 下载好游戏包后，往往会有一个.xci或者.nsp文件，在模拟器中点击“添加游戏目录”，然后将包含该.xci或者.nsp文件的上层文件夹作为游戏目录
  • 有时候会出现没有.xci或者.nsp的情况，这种情况下通常会包含一个.nsz的文件，这时候需要使用转换器进行格式转换（即将其转换为.xci或.nsp）
    • 下载SAK转换器
  • 关于补丁安装。模拟器左上角，"文件" -> “安装到文件NAND”，随后选择补丁文件即可

Mods下载

  • 模拟器自身提供了一个Mods库
  • Github上有个稳定的Mods项目
  • 上面两个链接提供的Mods数量较少，根据游戏，可以自行搜索一些Mods

Mods安装

  • 模拟器中，右键游戏条目，选择“打开MOD数据位置”
  • 依据如下的文件夹嵌套方式放置Mods文件
    • ...\yuzu\load\[一串很长的代码]\[mod的名字]\exefs
    • （下为例子）
    • ...\yuzu\load\0100F2C0115B6000\improved 60fps\exefs
